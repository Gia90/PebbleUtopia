## Main features:
* Cool battery indicator (Chilly: 100-80%, Sand: 70-50%, Bleach: 40-30%, Spoon: 20-10%, Eye: 10-0%)
* Customizable colors (Presets or custom)
* Customizable text (eg. Where is JESSICA HYDE?)
